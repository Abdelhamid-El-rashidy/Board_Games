\doxysubsection{Detailed Description}
User Interface class for the X-\/O (Tic-\/\+Tac-\/\+Toe) game. 

Inherits from the generic {\ttfamily Ufor player sI\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} base class and provides X-\/\+O{\ucr}specific functionality etup and move input.
