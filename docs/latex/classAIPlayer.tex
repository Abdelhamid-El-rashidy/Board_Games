\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classPlayer}{typename}} \mbox{\hyperlink{classPlayer}{T}}$>$\newline
class AIPlayer$<$ T $>$}
Base class for AI players following Open-\/\+Closed Principle (OCP). 

This class is open for extension but closed for modification. Derived classes only need to override\+:
\begin{DoxyItemize}
\item \doxylink{classAIPlayer_a7b61c8b977c9322cd5773309bc54c8e8}{create\+Move()} -\/ Create a move at given position
\item \doxylink{classAIPlayer_a38676c42e4a922eadace3633b5b046cb}{create\+Undo\+Move()} -\/ Create an undo move at given position
\end{DoxyItemize}

Optional overrides\+:
\begin{DoxyItemize}
\item \doxylink{classAIPlayer_a7a5e28485ca22a629601e98c3580b6d7}{condition()} -\/ Check if position is valid for a move (default\+: checks if cell is blank)
\item \doxylink{classAIPlayer_a706839a21ea012862d53269c478a56e6}{get\+\_\+opponent\+\_\+symbol()} -\/ Get opponent\textquotesingle{}s symbol (default\+: works for X/O games)
\item \doxylink{classAIPlayer_ab348ca15fad1cb2099d9a6336b41c637}{evaluate\+\_\+board()} -\/ Evaluate board position (default\+: simple heuristic)
\item \doxylink{classAIPlayer_a025987884475f11a556e327dc2a67aaa}{get\+\_\+max\+\_\+depth()} -\/ Maximum search depth (default\+: 3)
\item \doxylink{classAIPlayer_ab26d7531433c20c1616ec836a7fb5cce}{generate\+\_\+move\+\_\+values()} -\/ Generate possible move values for position (default\+: single move) 
\end{DoxyItemize}
