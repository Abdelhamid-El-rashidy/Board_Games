\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{classPlayer}{typename}} \mbox{\hyperlink{classPlayer}{T}}$>$\newline
class Board$<$ T $>$}
Base template for any board used in board games. 

Pure virtual function to undo the previous move played during backtracking algorithm. Must be implemented by derived classes.


\begin{DoxyTemplParams}{Template Parameters}
{\em T} & Type of the elements stored on the board (\doxylink{classPlayer_adb4e803a7ac0356b111b8ca79d766976}{e.\+g}., char, int, string).\\
\hline
\end{DoxyTemplParams}
Provides core data (rows, columns, matrix) and virtual methods to be implemented by specific games like Tic-\/\+Tac-\/\+Toe, Connect4, etc.
