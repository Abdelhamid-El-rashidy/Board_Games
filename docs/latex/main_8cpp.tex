\doxysubsection{Detailed Description}
Entry point for the FCAI X-\/O (Tic-\/\+Tac-\/\+Toe) game. 

This file initializes the X-\/O game by creating the user interface, board, and players. It then launches the game loop via the \doxylink{classGameManager}{Game\+Manager} class. All dynamically allocated objects are properly deleted at the end.
