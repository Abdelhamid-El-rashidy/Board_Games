\doxysubsection{Detailed Description}
User Interface class for the Misere (Tic-\/\+Tac-\/\+Toe) game. 

Inherits from the generic {\ttfamily \doxylink{classUI}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} base class and provides X-\/\+O{\ucr}specific functionality for player setup and move input.
