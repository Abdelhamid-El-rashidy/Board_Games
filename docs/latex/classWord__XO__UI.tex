\doxysubsection{Detailed Description}
User Interface class for the Word Tic-\/\+Tac-\/\+Toe game. 

Inherits from the generic {\ttfamily \doxylink{classUI}{UI}\texorpdfstring{$<$}{<}char\texorpdfstring{$>$}{>}} base class and provides Word X-\/O specific functionality for player setup and move input.
